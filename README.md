# 📊 Book Sales Analysis using SQL

## 🔹 Description

Analyzed an online bookstore dataset using SQL to understand customer behavior, order patterns, and sales performance.

## 🔹 Tables

* Books
* Customers
* Orders

## 🔹 SQL Queries Used

* Basic Queries (SELECT, WHERE, ORDER BY)
* Advanced Queries (JOIN, GROUP BY, Aggregate Functions)

## 🔹 Key Insights

* Identified top-selling books and genres
* Analyzed customer purchase behavior
* Calculated total revenue from orders
* Found most expensive and frequently ordered books

## 🔹 Tools

* MySQL
